3. Real-World Example: Navbar

A common use case combining justify-content and align-items.

Logo
  • Home
  • About
  • Contact
.navbar {
  display: flex;
  justify-content: space-between; /* Logo left, links right */
  align-items: center; /* Vertically centered */
}

.nav-links {
  display: flex;
  gap: 1rem;
}

4. Quick Reference

justify-content

Aligns items along main axis (horizontal)

align-items

Aligns items along cross axis (vertical)

align-self

Overrides align-items for individual item

align-content

Aligns flex lines (when wrapping)
