1. What is position: relative?

Relative positioning allows you to offset an element from its normal position without affecting other elements.

Key behavior: The element's original space is preserved in the document flow, but it visually appears offset.

4. Common Use Case: Positioning Context

The most common use of position: relative is to create a positioning context for absolutely positioned children.

Parent (position: relative)
Absolute
top: 0
right: 0
Absolute
bottom: 0
left: 0

This parent has position: relative, so its absolutely positioned children are positioned relative to it, not the viewport.

.parent {
  position: relative; /* Creates positioning context */
}

.child {
  position: absolute;
  top: 0;
  right: 0;
  /* Positioned relative to .parent, not viewport */
}
