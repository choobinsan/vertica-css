1. What are Grid Areas?

Grid template areas let you name sections of your grid and create layouts using those names, making your CSS more readable and maintainable.

Visual approach: Instead of using line numbers, you draw your layout using named areas!

2. Basic Syntax

.grid {
  display: grid;
  grid-template-areas:
    "header header header"
    "nav    main   aside"
    "footer footer footer";
  grid-template-columns: 1fr 3fr 1fr;
  grid-template-rows: auto 1fr auto;
}

header { grid-area: header; }
nav    { grid-area: nav; }
main   { grid-area: main; }
aside  { grid-area: aside; }
footer { grid-area: footer; }
ğŸ’¡ Tip: Each string represents a row, and each word represents a cell in that row.

6. Empty Cells with Dots

Use a dot (.) to create empty cells in your grid.

.grid {
  grid-template-areas:
    "header header header"
    ".      main   ."
    "footer footer footer";
  /* Middle cells on left and right are empty */
}

7. Responsive Layouts

Easily reorganize your layout for different screen sizes by changing grid-template-areas.

/* Desktop */
.grid {
  grid-template-areas:
    "header header"
    "nav    main"
    "footer footer";
}

/* Mobile */
@media (max-width: 600px) {
  .grid {
    grid-template-areas:
      "header"
      "nav"
      "main"
      "footer";
  }
}
ğŸ’¡ Pro tip: The same grid-area names work across breakpoints - just rearrange the template!
