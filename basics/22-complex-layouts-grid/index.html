1. Basic Spanning Example

Items can span multiple columns or rows using grid-column and grid-row.

1 (spans 2 columns)
2
3 (spans 2 rows)
4
5 (spans 2 columns)
6
7 (full width)
.item-1 {
  grid-column: 1 / span 2; /* Spans 2 columns */
}

.item-3 {
  grid-row: 2 / span 2; /* Spans 2 rows */
}

.item-7 {
  grid-column: 1 / -1; /* Full width */
}

  gap: 1rem;
}

.featured {
  grid-column: 1;
  grid-row: 1 / span 2; /* Tall featured article */
}
