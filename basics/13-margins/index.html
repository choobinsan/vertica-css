1. What are Margins?

Margins create space OUTSIDE an element's border, pushing other elements away.

Box with 20px margin
Box with 20px margin
.box {
  margin: 20px; /* All sides */
  margin-top: 10px;
  margin-right: 20px;
  margin-bottom: 10px;
  margin-left: 20px;
}

2. Centering with Margins

Use margin: auto to horizontally center block elements.

Centered with margin: auto
.centered {
  width: 300px;
  margin-inline: auto; /* Modern syntax */
  /* or margin: 0 auto; */
}
Key Point: Auto margins only work for horizontal centering on block elements with a defined width.

3. Margin Collapse

4. Parent-Child Margin Collapse

A child's margin can collapse with its parent's margin.

Child with margin-top: 20px

‚¨ÜÔ∏è The child's margin collapses through the parent!

.parent {
  background: lightgray;
  /* No padding or border */
}

.child {
  margin-top: 20px; /* Collapses with parent */
}

5. Preventing Margin Collapse

Several techniques can prevent margin collapse:

Child margin doesn't collapse

‚¨ÜÔ∏è Parent has padding, so no collapse!

/* Method 1: Add padding to parent */
.parent {
  padding-top: 1px;
}

/* Method 2: Add border to parent */
.parent {
  border-top: 1px solid transparent;
}

/* Method 3: Use flexbox or grid */
.parent {
  display: flex;
  flex-direction: column;
}
Note: Flexbox and Grid containers don't have margin collapse!

6. Negative Margins

Negative margins pull elements closer or create overlaps.

Box 1
Box 2 (margin-top: -20px)
.box {
  margin-top: -20px; /* Pulls element up */
}
Use with caution: Negative margins can cause overlapping content.
