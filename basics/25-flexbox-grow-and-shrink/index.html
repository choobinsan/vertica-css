4. How It Actually Works

flex-grow

Extra space is divided into total parts (sum of all flex-grow values).

Each item gets: (extra space / total parts) √ó its flex-grow value

Example: If extra space = 300px and ratios are 1:2:1
Total parts = 4, so each part = 75px
Item A gets 75px, Item B gets 150px, Item C gets 75px

flex-shrink

When items overflow, the deficit is distributed proportionally.

Items with higher flex-shrink values give up more space.

Example: flex-shrink: 0 means "never shrink me"
flex-shrink: 2 means "I'll shrink twice as fast as flex-shrink: 1"
