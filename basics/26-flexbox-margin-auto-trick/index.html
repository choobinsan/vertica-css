1. The Problem

You want to push one item to the opposite end of a flex container.

Common scenario: A navbar with logo on the left, links in the middle, and a button on the right.

2. The Solution: margin-left: auto

Use margin-left: auto to push an element to the right.

Logo

  • About
  • Contact

.navbar {
  display: flex;
  align-items: center;
}

.cta-button {
  margin-left: auto; /* Pushes button to the right! */
}
