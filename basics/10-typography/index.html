1. Font Family

Specify which typeface to use, with fallbacks.

This uses a serif font (Georgia)

This uses a sans-serif font (Arial)

This uses a monospace font (Monaco)

This uses the system font stack

.serif {
  font-family: Georgia, 'Times New Roman', serif;
}

.sans-serif {
  font-family: Arial, Helvetica, sans-serif;
}

.monospace {
  font-family: Monaco, 'Courier New', monospace;
}
Always provide fallbacks: List multiple fonts in case the first isn't available.

2. Font Size Units

Different units for sizing text, each with different behaviors.

16px - Fixed pixel size

1.2em - Relative to parent

1.2rem - Relative to root

120% - Percentage of parent

.px-size { font-size: 16px; }
.em-size { font-size: 1.2em; }
.rem-size { font-size: 1.2rem; }
.percent-size { font-size: 120%; }
Best practice: Use rem for most text sizing - it's relative to the root font size and more predictable than em.

3. Font Weight

Control the thickness/boldness of text.

100 - Thin

300 - Light

400 - Normal (default)

600 - Semi-bold

700 - Bold

900 - Black

.bold {
  font-weight: 700; /* or 'bold' */
}

.normal {
  font-weight: 400; /* or 'normal' */
}

4. Text Styling

Various properties to style text appearance.

font-style: italic

text-decoration: underline

text-decoration: line-through

text-transform: uppercase

text-transform: capitalize (each word)

5. Text Alignment

Control horizontal alignment of text.

text-align: left (default)

text-align: center

text-align: right

text-align: justify - This text is justified which means it stretches to fill the full width of the container, creating even edges on both sides.

6. Line Height & Spacing

Control vertical spacing and letter/word spacing.

line-height: 1.0

This text has tight line height. Notice how the lines are close together. This can make text harder to read.

line-height: 1.6

This text has normal line height. This is comfortable to read and is a common default value.

line-height: 2.0

This text has loose line height. The lines are spaced far apart, which can be useful for emphasis.

letter-spacing: 3px

Widely spaced letters

word-spacing: 10px

Widely spaced words here

.readable {
  line-height: 1.6;
  letter-spacing: 0.5px;
}
