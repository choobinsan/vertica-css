5. Can You Use Both?

Absolutely! They work great together.

Common pattern: Use Grid for the overall page layout, and Flexbox for components within grid items.

/* Grid for page layout */
.page {
  display: grid;
  grid-template-rows: auto 1fr auto;
}

/* Flexbox for header navigation */
header {
  display: flex;
  justify-content: space-between;
  align-items: center;
}

/* Grid for main content cards */
main {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 1rem;
}
