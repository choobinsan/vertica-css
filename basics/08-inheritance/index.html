1. What is Inheritance?

Child elements automatically inherit certain CSS properties from their parent elements.

This paragraph inherits color and font-family from its parent.

This nested paragraph also inherits from the parent.

.parent {
  color: #667eea;
  font-family: Georgia, serif;
}

/* Children automatically inherit color and font-family */

4. Overriding Inherited Values

Child elements can override inherited properties.

I inherit the blue color from parent

I override with my own green color

.parent {
  color: blue;
}

.child {
  color: green; /* Overrides inherited blue */
}
Key Point: Directly applied styles always override inherited styles.

5. The inherit Keyword

Force a property to inherit from its parent, even if it normally wouldn't.

This div inherits the border from its parent using border: inherit; 
.parent {
  border: 3px solid #667eea;
}

.child {
  border: inherit; /* Forces inheritance */
}
