CSS Layout Modes

Different ways to arrange elements on a page üìê

1. Block Layout

Block elements stack vertically and take full width.

Block Element 1
Block Element 2
Block Element 3
div {
  display: block; /* default for div */
  /* Takes full width available */
  /* Stacks vertically */
}
Block elements: div, p, h1-h6, section, article, header, footer, etc.

2. Inline Layout

Inline elements flow horizontally like text.

Inline 1 Inline 2 Inline 3 Inline 4 
span {
  display: inline; /* default for span */
  /* Only takes needed width */
  /* Flows horizontally */
  /* Width/height don't apply */
}
Inline elements: span, a, strong, em, img, button, etc.

3. Inline-Block

Flows like inline but respects width/height like block.

Box 1
Box 2
Box 3
.box {
  display: inline-block;
  width: 150px;
  height: 100px;
  /* Flows horizontally but respects dimensions */
}

4. Flexbox Layout

OChildren decide the layout with flexible sizing.

Flex 1
Flex 2
Flex 3
.container {
  display: flex;
  gap: 1rem;
  /* Children become flex items */
  /* Flexible, responsive layout */
}
Best for: Navigation bars, card layouts, centering, distributing space.

5. Grid Layout

Parent decides the layout with rows and columns.

Grid 1
Grid 2
Grid 3
Grid 4
Grid 5
Grid 6
.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  gap: 1rem;
  /* Creates a 3-column grid */
}
Best for: Page layouts, image galleries, complex multi-dimensional layouts.
