Grid Alignment & Justification

Control how grid items and tracks are positioned ⚖️

1. justify-items (Horizontal Alignment)

Aligns items horizontally within their grid cells.

.grid {
  display: grid;
  justify-items: center; /* start | center | end | stretch */

3. place-items (Shorthand)

Combines align-items and justify-items.

/* Shorthand for both */
.grid {
  place-items: center; /* Both axes */
  /* or */
  place-items: center start; /* align-items justify-items */
}

5. Quick Reference

justify-items

Horizontal alignment of items in cells

align-items

Vertical alignment of items in cells

place-items

Shorthand for both (align justify)

justify-content

Horizontal alignment of grid tracks

align-content

Vertical alignment of grid tracks

place-content

Shorthand for content (align justify)
