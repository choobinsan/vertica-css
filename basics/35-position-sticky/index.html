1. What is position: sticky?

Sticky elements behave like relative positioning until they reach a threshold, then become fixed.

Hybrid behavior: Sticky = Relative + Fixed. It's relative until you scroll past it, then it sticks!

2. How It Works

1

Scroll normally

Element behaves like position: relative

2

Reach threshold

When you scroll to the specified offset (e.g., top: 0)

3

Stick in place

Element becomes fixed at that position

4

Scroll back

Returns to relative positioning

3. Basic Syntax

.sticky-element {
  position: sticky;
  top: 0; /* Stick when top reaches 0px from viewport */
  /* Element stays in document flow */
}

7. Key Differences

Fixed
Always fixed to viewport, removed from flow
Sticky
Toggles between relative and fixed, stays in flow
Relative
Always relative, never sticks
