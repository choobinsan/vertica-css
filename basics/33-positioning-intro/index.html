1. Static Positioning (Default)

Elements flow in normal document order. Top, right, bottom, left properties have no effect.

position: static

This is the default. Elements appear in normal flow.

.element {
  position: static; /* default */
}

2. Relative Positioning

Element is positioned relative to its normal position. Original space is preserved.

Normal box
position: relative

Moved 20px down and 20px right from normal position

top: 20px; left: 20px;
Notice the gap above
Key Point: The element's original space is still reserved in the document flow.

3. Absolute Positioning

Element is removed from normal flow and positioned relative to its nearest positioned ancestor.

Positioned Parent (position: relative)
Normal box
position: absolute

Positioned relative to parent

top: 20px; right: 20px;
Normal box
Key Point: Element is removed from flow. Other elements act as if it doesn't exist.

4. Fixed Positioning

Element is positioned relative to the viewport. Stays in place when scrolling.

Look at the bottom-right corner! There's a fixed box that stays visible as you scroll. 
.element {
  position: fixed;
  bottom: 20px;
  right: 20px;
}

5. Sticky Positioning

Hybrid of relative and fixed. Element is relative until it reaches a threshold, then becomes fixed.

position: sticky

6. Quick Reference

static

Default flow

Use: Default behavior

relative

Offset from normal position

Use: Small adjustments, anchor for absolute children

absolute

Removed from flow

Use: Overlays, tooltips, dropdowns

fixed

Relative to viewport

Use: Headers, floating buttons

sticky

Relative + fixed hybrid

Use: Table headers, navigation
