1. Type Selector (Element Selector)

Selects all elements of a given type. Use the HTML tag name.

This paragraph is styled using a type selector.

All paragraphs with this class get the same styling.

p {
  color: blue;
  font-size: 16px;
}

2. Class Selector

Selects all elements with a specific class attribute. Prefix with a dot (.).

This div uses a class selector
Multiple elements can share the same class
.class-selector-example {
  background: #667eea;
  color: white;
  padding: 1rem;
}

3. ID Selector

Selects a single element with a specific ID attribute. Prefix with a hash (#). IDs should be unique on a page.

This div uses an ID selector
#id-selector-example {
  background: #764ba2;
  color: white;
  padding: 1rem;
}
Best Practice: Use classes for styling, reserve IDs for JavaScript hooks or unique page anchors.

4. Universal Selector

Selects all elements. Use the asterisk (*).

All elements inside this container

are affected by the universal selector 
* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}
Performance Note: The universal selector can be slow on large pages. Use it sparingly.
