1. The Problem with Images

By default, images can overflow their containers.

Container (300px wide)

‚ö†Ô∏è Problem: The image is wider than its container and overflows!

2. Responsive Images

Use max-width: 100% to make images responsive.

img {
  max-width: 100%;
  height: auto; /* Maintains aspect ratio */
  display: block;
}
How it works: The image will never be wider than its container, but can be smaller.

3. Modern Syntax

Use logical properties for better internationalization.

img {
  max-inline-size: 100%; /* Modern */
  block-size: auto;

  /* vs older syntax */
  max-width: 100%;
  height: auto;
}
Note: max-inline-size adapts to writing direction (LTR/RTL).

4. Object-fit Property

Control how images fill their containers.

object-fit: fill

object-fit: contain

object-fit: cover

object-fit: none

.avatar {
  width: 200px;
  height: 200px;
  object-fit: cover; /* Fills container, crops if needed */
}

5. Object-position

Control which part of the image is visible when cropped.

object-position: top

object-position: center

object-position: bottom
