1. Compound Selectors

Combine element and class selectors (no space) to target specific elements with a class.

Main Title (h3.main-title)

Secondary Title (h3.secondary-title)

Paragraph with main-title class (not selected)

/* Selects only h3 elements with class "main-title" */
h3.main-title {
  color: #667eea;
}

/* Selects only h3 elements with class "secondary-title" */
h3.secondary-title {
  color: #764ba2;
}
Key Point: No space between selectors means "element WITH this class". The paragraph with class "main-title" is NOT selected because it's not an h3.

2. Descendant Selectors

Use a space between selectors to target elements inside other elements (at any depth).

Direct child paragraph

Nested paragraph (inside inner-container)

Deeply nested paragraph

/* Selects ALL p elements inside .container (any depth) */
.container p {
  background: #f0f4ff;
  padding: 1rem;
}

/* More specific: p inside .inner-container */
.inner-container p {
  border: 2px solid #667eea;
}
Key Point: Descendant selectors work at ANY depth. All paragraphs inside .container are selected, whether direct children or deeply nested.

3. Combining Both

You can combine compound and descendant selectors for very specific targeting.

Highlighted paragraph in special container

Regular paragraph in special container

Nested highlighted paragraph

/* Selects p elements with class "highlight" inside .special-container */
.special-container p.highlight {
  background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
  color: white;
}
