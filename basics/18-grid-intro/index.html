1. Using fr Units

The fr unit represents a fraction of available space in the grid container.

1fr 2fr 1fr (middle column gets twice the space)
1fr
2fr

.grid-container {
  display: grid;
  grid-template-columns: 1fr 2fr 1fr;
  gap: 15px;
}
How it works: Total is 4fr (1+2+1). First column gets 25%, middle gets 50%, last gets 25%.

2. Using repeat()

The repeat() function creates equal columns without repetition.

repeat(3, 1fr) - three equal columns

.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 15px;
}
Shorthand: repeat(3, 1fr) is the same as 1fr 1fr 1fr

3. Mixed Units

Combine fixed widths (px) with flexible units (fr) for powerful layouts.

100px 1fr 2fr (fixed sidebar + flexible content)
100px
1fr
2fr
.grid-container {
  display: grid;
  grid-template-columns: 100px 1fr 2fr;
  gap: 15px;
}
Common pattern: Fixed sidebar (100px) + flexible content areas that share remaining space.
