1. What are Intrinsic Patterns?

Intrinsic patterns use CSS properties that automatically adapt to available space, eliminating the need for media queries.

Key benefit: The layout responds to its container size, not just the viewport. This makes components truly reusable!

2. auto-fit Pattern

4. The RAM Pattern

Repeat, Auto-fit, Minmax - a powerful combination!

/* The RAM pattern */
.grid {
  display: grid;
  grid-template-columns:
    repeat(auto-fit, minmax(min(250px, 100%), 1fr));
  gap: 1rem;
}
Why min(250px, 100%)? Prevents overflow on very small screens by allowing columns to shrink below 250px if needed.

/* Automatically stacks when sidebar can't fit */
