1. Why CSS Resets?

Different browsers apply different default styles. Resets create a consistent baseline.

The Problem: Browsers add default margins, padding, and styles that vary between Chrome, Firefox, Safari, etc.

2. Common Default Styles

Examples of browser defaults that resets remove:

body

Has default margin (usually 8px)

h1, h2, p

Have default margins

ul, ol

Have padding and list-style

button

Has browser-specific styling

3. Simple Reset Example

A minimal reset to remove common defaults:

/* Simple CSS Reset */
* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

/* Better approach - preserve useful defaults */
*,
*::before,
*::after {
  box-sizing: border-box;
}

body {
  margin: 0;
  line-height: 1.5;
}

h1, h2, h3, h4, h5, h6, p {
  margin: 0;
}

ul, ol {
  list-style: none;
  padding: 0;
}
