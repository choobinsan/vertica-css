5. Line-Based Placement

Specify exact start and end lines instead of span.

1 to 3
3 to 5
1 to 5
/* Grid has 4 columns = 5 lines */
.item {
  grid-column: 1 / 3; /* Start at line 1, end at line 3 */
  /* Same as: grid-column: span 2; */
}

.full-width {
  grid-column: 1 / 5; /* Spans all 4 columns */
  /* Same as: grid-column: span 4; */
}
Grid lines: A 4-column grid has 5 vertical lines (1, 2, 3, 4, 5).

6. Shorthand Syntax

grid-column: span 2;

Span 2 columns from current position

grid-column: 1 / 3;

From line 1 to line 3

grid-column: 1 / -1;

From first to last line (full width)

grid-row: span 2;

Span 2 rows

grid-area: 1 / 1 / 3 / 3;

row-start / col-start / row-end / col-end
