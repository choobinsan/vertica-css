1. What is @font-face?

The @font-face rule allows you to load custom fonts from a server and use them in your CSS.

Why use it? You're not limited to web-safe fonts. Load any font you want!

2. Basic Syntax

@font-face {
  font-family: 'MyCustomFont';
  src: url('fonts/mycustomfont.woff2') format('woff2'),
       url('fonts/mycustomfont.woff') format('woff');
  font-weight: normal;
  font-style: normal;
}

body {
  font-family: 'MyCustomFont', sans-serif;
}

3. Font Formats

WOFF2

Best choice

Modern, compressed, widely supported

WOFF

Fallback

Good support, slightly larger

TTF/OTF

Legacy

Older browsers, larger file size

üí° Recommendation: Use WOFF2 as primary, WOFF as fallback.

4. Using Google Fonts

The easiest way to use custom fonts is with Google Fonts.

Method 1: Link Tag

<link rel="preconnect" href="https://fonts.googleapis.com">
<link href="https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap" rel="stylesheet">

Method 2: @import

@import url('https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap');

5. Font Weights & Styles

Define different weights and styles for the same font family.

/* Regular */
@font-face {
  font-family: 'MyFont';
  src: url('myfont-regular.woff2') format('woff2');
  font-weight: 400;
  font-style: normal;
}

/* Bold */
@font-face {
  font-family: 'MyFont';
  src: url('myfont-bold.woff2') format('woff2');
  font-weight: 700;
  font-style: normal;
}

/* Italic */
@font-face {
  font-family: 'MyFont';
  src: url('myfont-italic.woff2') format('woff2');
  font-weight: 400;
  font-style: italic;
}

6. Font Display Property

Control how fonts are displayed while loading.

font-display: swap;

Recommended - Show fallback immediately, swap when loaded

font-display: block;

Hide text briefly, then show custom font

font-display: fallback;

Brief block period, swap if loaded quickly

font-display: optional;

Use custom font only if already cached

@font-face {
  font-family: 'MyFont';
  src: url('myfont.woff2') format('woff2');
  font-display: swap; /* Recommended for performance */
}
