1. What is position: fixed?

Fixed elements are removed from the normal flow and positioned relative to the viewport.

Key behavior: Fixed elements stay in the same position even when scrolling.

2. Common Use Cases

üîù Fixed Header

Navigation that stays at the top while scrolling

3. Basic Syntax

.fixed-element {
  position: fixed;
  top: 0;
  right: 0;
  /* Positioned relative to viewport */
  /* Stays in place when scrolling */
}
