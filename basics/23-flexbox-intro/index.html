Introduction to Flexbox

Flexbox is a one-dimensional layout system for arranging items in rows or columns üìê

1. Basic Flexbox Container

Add display: flex to a container to make its children flex items.

Default behavior (no wrap)
1
2
3
4
5
6
.flex-container {
  display: flex;
  /* flex-wrap: nowrap; (default) */
}
What's happening? Items shrink to fit on one line. Try resizing your browser window!

2. Flex Wrap

Use flex-wrap: wrap to allow items to wrap to multiple lines.

With wrapping enabled
1
2
3
4
5
6
.flex-container {
  display: flex;
  flex-wrap: wrap;
}
What's happening? Items maintain their size and wrap to new lines when needed.

3. Flex Direction

Control the direction items flow: row (default) or column.

Column direction
1
2
3
.flex-container {
  display: flex;
  flex-direction: column;
}

4. Key Flexbox Properties

display: flex

Creates a flex container

flex-direction

row | row-reverse | column | column-reverse

flex-wrap

nowrap | wrap | wrap-reverse

gap

Space between flex items
