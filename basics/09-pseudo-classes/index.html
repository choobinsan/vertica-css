1. Interactive States

Style elements based on user interaction.

Hover over me! 
a:hover {
  color: #e74c3c;
  transform: scale(1.1);
}

button:active {
  transform: scale(0.95);
}

input:focus {
  border-color: #667eea;
  box-shadow: 0 0 0 3px rgba(102, 126, 234, 0.2);
}
Common states: :hover, :active, :focus, :visited, :disabled

2. Structural Pseudo-classes

Target elements based on their position in the document.

  • First item (:first-child)
  • Second item (:nth-child(2))
  • Third item
  • Fourth item
  • Last item (:last-child)
li:first-child {
  font-weight: bold;
  color: #27ae60;
}

li:last-child {
  font-style: italic;
  color: #e74c3c;
}

li:nth-child(2) {
  background: #fff3cd;
}

/* Odd items: 1, 3, 5, 7... */
.item:nth-child(odd) {
  background: #e3f2fd;
}

/* Even items: 2, 4, 6, 8... */
.item:nth-child(even) {
  background: #f3e5f5;
}

/* Every 3rd item: 3, 6, 9... */
.item:nth-child(3n) {
  background: #fff3cd;
}

4. Form Pseudo-classes

Style form elements based on their state.

Try leaving this empty 
input:required {
  border-left: 3px solid #e74c3c;
}

input:valid {
  border-color: #27ae60;
}

input:invalid {
  border-color: #e74c3c;
}

input:disabled {
  opacity: 0.5;
  cursor: not-allowed;
}

input:checked {
  accent-color: #667eea;
}

5. Quick Reference

:last-child

Last child element

Nth child element

:not(selector)
