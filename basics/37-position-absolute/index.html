1. What is position: absolute?

Absolute positioning removes an element from the normal document flow and positions it relative to its nearest positioned ancestor.

Key behavior: The element is removed from the flow (doesn't take up space) and positioned relative to the nearest ancestor with position: relative, absolute, fixed, or sticky.

5. Centering with Absolute

A classic technique for centering elements.

Perfectly Centered!
.centered {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
  /* Centers element perfectly */
}

6. Stretching to Fill

Set all four properties to 0 to stretch an element to fill its parent.

Stretched to fill parent
(top, right, bottom, left all = 0) 
.stretched {
  position: absolute;
  top: 0;
  right: 0;
  bottom: 0;
  left: 0;
  /* Fills entire parent */
}
