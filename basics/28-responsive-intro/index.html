1. What is Responsive Design?

Responsive design ensures your website looks good and functions well on all devices - from phones to tablets to desktops.

Core principle: One website that adapts to different screen sizes, rather than separate mobile and desktop sites.

2. The Viewport Meta Tag

Essential for responsive design - tells browsers how to handle page dimensions.

<meta name="viewport" content="width=device-width, initial-scale=1.0">
üí° Always include this: Without it, mobile browsers will zoom out to show the full desktop site.

3. Flexible Images

Images should scale with their container.

This image scales with the container (try resizing your browser)

  max-width: 100%;
  height: auto;
  display: block;
}

7. Mobile-First Approach

Start with mobile styles, then add complexity for larger screens.

‚úÖ Mobile-First (Recommended)

/* Base: Mobile */
.container {
  padding: 1rem;
}

/* Add for larger screens */
@media (min-width: 600px) {
  .container {
    padding: 2rem;
  }
}

‚ùå Desktop-First

/* Base: Desktop */
.container {
  padding: 2rem;
}

/* Override for smaller screens */
@media (max-width: 600px) {
  .container {
    padding: 1rem;
  }
}
