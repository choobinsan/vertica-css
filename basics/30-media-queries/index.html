1. What are Media Queries?

Media queries allow you to apply CSS only when certain conditions are met (like screen width, orientation, or resolution).

@media (min-width: 600px) {
  /* Styles for screens 600px and wider */
  .container {
    display: grid;
    grid-template-columns: repeat(2, 1fr);
  }
}
How it works: The styles inside the @media block only apply when the condition is true.

4. Combining Conditions

Use logical operators to combine multiple conditions.

/* AND: Both conditions must be true */
@media (min-width: 600px) and (max-width: 900px) {
  /* Styles for tablets only */
}

/* OR: At least one condition must be true */
@media (orientation: landscape), (min-width: 1200px) {
  /* Styles for landscape OR large screens */
}

/* NOT: Condition must be false */
@media not (orientation: portrait) {
  /* Styles for non-portrait (landscape) */
}

7. Print Styles

Media queries aren't just for screen sizes - you can target print too!

@media print {
  /* Hide navigation when printing */
  nav {
    display: none;
  }

  /* Ensure good contrast */
  body {
    color: black;
    background: white;
  }

  /* Show link URLs */
  a::after {
    content: " (" attr(href) ")";
  }
}
