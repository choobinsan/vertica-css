1. Color Keywords

Use predefined color names like red, blue, green, etc.

red
blue
green
purple
.element {
  background-color: red;
  color: white;
}
Note: There are 147 named colors in CSS. Easy to remember but limited palette.

2. Hexadecimal Colors

Use hex codes with # followed by 6 digits (RRGGBB) or 3 digits shorthand.

#FF0000 Red
#0000FF Blue
#00FF00 Green
#667eea Purple 
/* Full form */
.element {
  background-color: #FF0000;
}

/* Shorthand (when pairs match) */
.element {
  background-color: #F00; /* same as #FF0000 */
}
How it works: #RRGGBB where RR=red, GG=green, BB=blue (00-FF in hexadecimal)

3. RGB Colors

Define colors using Red, Green, Blue values (0-255).

rgb(255, 0, 0) Red
rgb(0, 128, 0) Green
rgb(0, 0, 255) Blue
rgb(102, 126, 234) Purple 
.element {
  background-color: rgb(255, 0, 0);
  /* R=255, G=0, B=0 */
}

4. RGBA Colors (with Transparency)

RGB plus Alpha channel for transparency (0.0 = transparent, 1.0 = opaque).

rgba(255, 0, 0, 1.0) 100% Opaque
rgba(255, 0, 0, 0.7) 70% Opaque
rgba(255, 0, 0, 0.4) 40% Opaque
rgba(255, 0, 0, 0.1) 10% Opaque 
.element {
  background-color: rgba(255, 165, 0, 0.7);
  /* Orange with 70% opacity */
}

6. HSLA Colors (with Transparency)

HSL plus Alpha channel for transparency.

hsla(240, 100%, 50%, 1.0) 100% Opaque
hsla(240, 100%, 50%, 0.7) 70% Opaque
hsla(240, 100%, 50%, 0.4) 40% Opaque
hsla(240, 100%, 50%, 0.1) 10% Opaque
